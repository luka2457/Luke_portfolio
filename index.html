Portfolio

WePlay

WePlay is a platform for people to find, join or create pickup sports games.The application was built using the MERN stack, consisting of a Node/Express backend, a Mongo database and React frontend. Google Maps displays event location and Nodemailer allows users to contact or share an event.

Take A Hike

This application utilizes REI's hiking API as well as a OpenWeatherMap API to provide popular hikes and current weather upon users zip code entry. Firebase is used to store searches and provide user with recent searches. The application is mobile friendly with the use of CSS Grid styling.

RYTFA

A website for The Rochester Youth Track and Field Association. The site was converted from a Ruby on Rails website (found here) to an exact static replica with the use of only Javascript and Bootstrap. Application is deployed and hosted through Amazon Web Services.

The Office Trivia

For all fellow "The Office" nerds out there. This trivia game makes use of Javascripts ability to dynamically maneuver between questions based on time while also using jQuery to generate new HTML.

PlatePlanner

Full stack application that allows users to search and save recipes as well as ingredients to their profile. Front end operates via JS with the help jQuery and SASS for styling. The site is built on Node and utilizes Sequelize to save client profile information.

Guess That 14er

Guess that 14er is a hangman-like word guessing game built with vanilla Javascript and bootstrap styling. In the application you will be provided a picture of a 14er and get a few chances to guess the mountain correctly. Good luck!
